■　はじめに

「マルチメディアライブラリ・マネージャ(MMLib)」をダウンロードしていただき、ありがとうございます。

オフィスでも自宅でも，捨てるに捨てられない雑誌記事や書類などがたくさんあります．それらを自分のパソコンに納めてしまうことも，最近のパソコンの進歩が可能にしてくれます．そこから「モノは捨てるけれど情報は捨てない」という新しいライフスタイルが見えてきます．
そのためのツールが「マルチメディアライブラリ」です．「電子ファイリング」というと，ちょっと大げさかもしれません．自分の周りの画像情報を手軽に整理・保管・配布するためのツールです．

「マルチメディアライブラリ」はつぎのような機能をもっています。

・画像以外にもテキストデータを扱うこともできます
・TWAIN対応のイメージスキャナやデジタルカメラから直接画像を入力できます
・ビデオカメラの画像を取り込むこともできます（静止画のみ）
・パスワード設定およびアクセス制限によるセキュリティ機能
・高速なキーワード検索機能
・任意の他の画像ソフト（ＯＣＲソフト、レタッチソフト等）との連携をサポート（データ　転送機能）
・専用画像ビューワ(MMView)を用意
・MS Outlookとの連携をサポート（メール送信機能）

・ネットワーク版ではリモートサーバーのHDD上で画像情報を共有することができます
・ネットワーク版では一度読み込んだ画像をキャッシュに保存することができます
　（ネットワーク版については別途お問い合わせください）

■　インストール方法

MMLibJ.exeを実行してください。

１．InstallShieldウィザードが立ち上がりますので「次へ」をクリックします。
２．使用許諾契約書が表示されますので、画面をスクロールして全文お読みいただいた上で「使用許諾契約の条項に同意します」をクリックすると「次へ」を押すことができるようになります。同意いただけない場合はここでインストールを中断してください。
３．「ユーザ名」と「所属」を入力して（あるいは確認して）「次へ」を押してください。
４．インストール先のフォルダはC:\Program Files\Multimedia Libraryになっています。他のフォルダにインストールしたい場合は右の「変更」ボタンを押してインストール先フォルダを指定してください。「次へ」を押します。
５．インストールの設定内容が表示されます。確認のうえ「インストール」ボタンを押してください。インストールが開始されます。
６．「InstallShieldウィザードを完了しました」と表示されたならインストールは完了です。「完了」ボタンを押してダイアログを閉じてください。

■　動作確認

「スタート」メニューの「Multimedia Library」から「Multimedia Library Manager」を選択して起動してください。本プログラムは３０日間、試用することができます。ダイアログから「試用」ボタンを押してください。

▼ Twain対応スキャナをお持ちの場合

１．「MMLib」の「ファイル」メニューの「環境設定」ダイアログの「Twainドライバ」タグを選択して、接続されているスキャナ用のドライバが選択されていることを確かめてください。複数のドライバがインストールされている場合は「ドライバ選択」ボタンを押して、お使いになるドライバを選んでください。
２．メインウィンドウ上部の「新規データ」ボタンをクリックし、表示されたダイアログ右下の「スキャナ」ボタンを押してください。スキャナの読み取りプログラムが起動しますので手近な写真などを読み込んでみてください。（読み取り作業はMMLibではなくスキャナ用プログラムで行います）
３．１枚取り込むと元のダイアログに戻ってくるので、任意の「名称」をつけて「保存」ボタンを押してから「閉じる」ボタンを押してください。
４．メインウィンドウの「インボックス」の下に「Inbox＋10桁の数字」（数字は月日時分）のフォルダが作られ、その中にさきほどスキャンした画像が入っているはずです。以上でMMLibは正常に動作しています。

▼ スキャナをお持ちでない場合

１．メインウィンドウ上部の「新規データ」ボタンをクリックし、表示されたダイアログ右下の「ファイル読込」ボタンを押してください。
２．「画像ファイルの選択」ダイアログが表示されるので、BMP/GIF/JPEGのいずれかの形式の画像ファイルを選んで「開く」ボタンを押してください。
３．元のダイアログに戻ってくるので、任意の「名称」をつけて「保存」ボタンを押してから「閉じる」ボタンを押してください。
４．メインウィンドウの「インボックス」の下に「Inbox＋10桁の数字」（数字は月日時分）のフォルダが作られ、その中にさきほど読み込んだ画像が入っているはずです。以上でMMLibは正常に動作しています。
