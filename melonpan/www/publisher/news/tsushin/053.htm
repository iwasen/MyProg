　こんにちは。めろんぱん事務局の松下　友見子です。

　長かった夏も終わり、ここ東京では少し涼しい風が吹いたりしています。
　この夏の私は、休みといえば趣味のテニスをして、
　汗をかく機会が多かった気がします。

　疲れをとるために、寝るのはもちろん大事なのですが、
　私の場合、たまに「もうだめぇ〜」というまで運動した方が、
　体が調子良く過ごせることに気付いてしまいました。

　ということで、私の秋は「スポーツの秋」になりそうです。

　みなさんは、今年はどんな秋になりそうですか？

　めろんぱんはサーバメンテナンスを行います。

┗◆┓めろんぱんの想い
　┗┛────────────────────────────────

>━━ 発行していて嬉しかったことはなに？　苦労することはなに？

　「めろんぱん」では、本登録から１年経ったメルマガの発行者さんへメー
　ルをお届けしております。これは、１年間めろんぱんでご発行いただいた
　ことへの感謝と、なにかの区切りとして感じていただき、また２年目もが
　んばって発行していただきたいという想いから、お送りしているものです。

　また、１年間ご発行いただいた発行者の方のご意見を伺って少しでも今後
　の運営の参考にさせていただければと思い、そのメールの中でちょっとし
　たアンケートをとらせていただいています。

　項目は「メルマガの良いところは？」「発行していて嬉しかったことは？」
　などというものです。

　みなさんのコメントを拝見していると、やはり日ごろの体験からの生々し
　い言葉が並び、非常に説得力のある言葉だなと思います。

　そこで今回、みなさんのコメントの中から、キーワードを少しまとめてみ
　ましたので、ご紹介いたします。


　　■メルマガを発行していて良かったこと・嬉しかったこと
　　│
　　├◇読者との関係
　　│　　　読者から感想がくる
　　│　　　読者の共感が得られる
　　│　　　読みつづけてくれる読者がいるということ
　　│　　　メルマガを通して友人ができる
　　│
　　├◇自分にとって
　　│　　　自由に書ける
　　│　　　知人に近況を伝えるツールにもなる
　　│
　　└◇嬉しかったこと
　　　　　　読者が増えたこと
　　　　　　読者から成果報告があった
　　　　　　メルマガの目標としていることが実現できている

　………みなさん、さまざまな経験をされていらっしゃいますね。
　　　　自分が好きで表現したことに、読者の方が反応してくれる、
　　　　他のメディアでは、なかなか得るのが難しいものであり、これが、
　　　　メルマガの良さなのだろうと思います。

　　　　めろんぱんでも、より読者の方とのコミュニケーションがとりやす
　　　　いよう、発行者管理ページにて「読者の広場をつくる」という
　　　　メニューをご用意しています。よろしければ、こちらもぜひご利用
　　　　ください。
　　　　 http://www.melonpan.net/publisher/hakkou-login.html

　　　　また、「読者から成果報告」を喜びとしているかたもいて、メルマ
　　　　ガを発行する喜びも、そのメルマガによって様々なのだということ
　　　　も改めて実感しました。


　　■メルマガを発行していて大変な事・苦労すること
　　│
　　├◇書きつづけるということ
　　│　　　発行ペースを守る
　　│　　　毎回同じ質・量を保つ
　　│　　　ネタ集め
　　│
　　├◇具体的な作業で
　　│　　　ＰＣに不慣れである
　　│　　　（投稿）情報の整理
　　│
　　├◇コンテンツ作成
　　│　　　わかり易さ・公平さへの配慮
　　│　　　著作権や肖像権への配慮
　　│　　　誹謗中傷にならないようにする
　　│
　　└◇読者との関係
　　　　　　簡単に解除される
　　　　　　一方通行であることがつらい

　………やはり、書きつづけるということが一番大変なことなのかも知れま
　　　　せん。それでも、コンテンツの充実には手を抜かずに、多くのこと
　　　　を心がけながら発行されているからこそ読者の方を惹きつけている
　　　　のだろうと思います。

　　　　具体的な発行作業の面などでは、めろんぱんでもまだまだお手伝い
　　　　できることがあるかもしれません。そんなときには、ぜひ事務局まで
　　　　お知らせくださいませ。

                      (#)　　　　(#)　　　　(#)

　１年間発行すると、いろいろな体験、苦労、喜びがあるということことが
　伝わってくる結果だと思います。メルマガ発行者歴が長い方も、最近発行
　を始めた方も、なにかの参考になればいいなと思います。

　このアンケートでは、「メルマガで工夫すること」などもうかがっていま
　す。そちらは次号以降で、またご紹介させていただきたいと思います。
　どうぞお楽しみに！
