　こんにちは。めろんぱん事務局の松下　友見子です。

　１０月になると、色々な行事がありますよね。
　個人的には、小学校の運動会を思い出します。

　私は体がちっちゃかったので、騎馬戦の「上で戦う人」だったのですが、
　担いでくれた友人たちがあまりのスピードで突進したため、
　上から落ちて、みんなで転んだことを鮮明におぼえています。
　今ではいい思い出ですが。

　そんな結構過激で（！？）楽しい運動会は
　今の時代も、おこなわれているのでしょうか？？

┌──────────────────────────────────
■１１月７日１４：００〜１５：００の間、サーバメンテナンスのため、
│めろんぱんは全てのサービスを停止いたします。
└──────────────────────────────────

　来月のお話になりますが、１１月７日（木）１４：００〜１５：００の間、
　めろんぱんはサーバメンテナンスを行います。

　この間はめろんぱんの全ての機能を停止させていただくことになります。
　恐れ入りますが、この時間帯へのご予約、及び、この時間帯での発行作業
　はできませんので、ご注意いただきますようお願いいたします。

　日頃ご発行くださっている発行者のみなさまには、大変ご迷惑をお掛けい
　たしますが、何卒、ご了承くださいますようお願いいたします。

　また、安定したシステムをご提供させていただくために、今後も定期的に
　今回のような短時間のメンテナンスを行っていく予定でおります。
　どうぞよろしくお願いいたします。


┌──────────────────────────────────
■“第６代・メルマガナビ”を募集いたします。
└──────────────────────────────────

　このたび、６代目の「メルマガナビ」を募集させていただくことになりま
　した。

　この「メルマガナビ」はご存知でしょうか？
　めろんぱんで、多くのメルマガを読んでいただき、
　　　・「オススメメルマガ」を選んだり
　　　・読者からの「こんなメルマガ探して！」という依頼に答えたり
　　　・各メルマガに届く「推薦文」をチェックしたり
　というようなお仕事しています。

　そのメルマガナビをこのたび３名募集いたします。
　「メルマガが大好き」な方で、多くのメルマガを読んでいただける方に、
　ぜひご応募いただければ嬉しいです。

　もちろん、お仕事としてお願いしておりますので、わずかではありますが
　報酬もご用意しております。

　関心がおありの方は、下記のアドレスまでぜひご連絡ください。
　待遇等、追って詳細をお知らせいたします。

　◎宛先：めろんぱん事務局「ナビ募集受付係」 melonpai@melonpan.net
　　　　　※ご発行中のメルマガ名と、発行者名を明記してください。
　◎締切：１０月７日（月）午前１０時

　たくさんの方のご応募を楽しみにお待ちしております。

┗◆┓めろんぱんの想い
　┗┛────────────────────────────────

>━━ メルマガを発行する上で、工夫していることはなんですか？

　前号の「めろんぱん通信」では、発行１年を迎えた発行者さんたちの「発
　行していて嬉しかったこと」「苦労すること」などをご紹介させていただ
　きました。
　（発行者用管理ページの左上「めろんぱん通信　Back Number」よりご覧い
　　ただけます。 http://www.melonpan.net/publisher/hakkou-login.html ）

　今回は、「工夫していること」についてご紹介したいと思います。


　　■メルマガを発行する上で、工夫していることはなんですか？
　　│
　　├◇質の維持・向上のために
　　│　　　多くの人の声を聞く
　　│　　　写真や画像で内容の補足をする
　　│　　　実用性がある情報にする
　　│　　　ほかにはない情報提供を心がける
　　│　　　情報の正確さを心がける
　　│　　　何人かのライターが合同でつくる
　　│　　　校正をくりかえす
　　│
　　├◇受け取る読者のことを考えて
　　│　　　飽きないような内容を心がける
　　│　　　読みやすいレイアウトにする
　　│　　　読者からのアクションを受け取りやすくする
　　│　　　広告を少なく抑える
　　│　　　イベント企画
　　│　　　ちょうどよいボリューム
　　│　　　途中から読む読者にもわかるようにする
　　│　　　呼びかけるような文体にする
　　│　　　手紙のような文体にする
　　│
　　└◇自分自身について
　　　　　　自分が楽しめるように


　………「正確な情報」であったり、「オリジナリティのあるコンテンツ」
　　　　をみなさん心がけていらっしゃるようですね。また、最近では個人
　　　　で発行している方々が集まって、１つのメルマガを作りあげる、な
　　　　んていうメルマガも結構みかけます。発行者さんたちのそんな挑戦
　　　　も応援しています。

　　　　２番目の項目の中での「途中から読む読者にもわかるようにする」
　　　　なんていうところは、小さなことですが、読者にとっても嬉しい工
　　　　夫ですね。たとえ物語のように毎回継続したお話のメルマガではな
　　　　くても、ちょっとした発行者の方の自己紹介などが毎回ついている
　　　　と途中から読む読者の方も楽しめるのではないかなと思います。よ
　　　　ろしければ、ぜひみなさんも参考になさってくださいね。

　　　　最後の「自分が楽しめるように」ですが、実はコレ、すごく大事な
　　　　ことですよね。“もともと好きで発行していた”のに、継続して発
　　　　行しているうちに義務感が強くなったり発行周期に追われて“書く
　　　　こと自体楽しめていない”という声も耳にすることがあります。
　　　　「自分が楽しんで書く」ことは、きっとコンテンツにも表れ、読者
　　　　の方にも楽しんでもらえる大事な要素なんだろうな、と思いました。

                      (#)　　　　(#)　　　　(#)
